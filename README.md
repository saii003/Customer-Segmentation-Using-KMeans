# Customer-Segmentation-Using-KMeans
Machine Learning project for customer segmentation using K-Means clustering.


## Project Overview

This project segments customers into different groups based on Annual Income and Spending Score using the K-Means Clustering algorithm.

The segmentation helps businesses identify different customer groups for targeted marketing and business strategies.

---

## Features

- Data Preprocessing
- Customer Segmentation
- K-Means Clustering
- Elbow Method
- Data Visualization
